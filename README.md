# rigidBodySim
### Overview:
This is a simple and incomplete Rigid body Simulator written in C, using SDL2.


### Installing and Running:
Install SDL2
1. Clone Repository and Make
    ```bash
    git clone https://github.com/LukeW2002/rigidBodySim.git
    cd nBodySimulator
    make && make clean
    ```
2. Run the program
   ```bash
   ./orbits
   ```
